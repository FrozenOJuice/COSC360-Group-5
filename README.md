Steps to run through docker:
Create a .env file in backend/ and paste the variables

Open a new terminal
In the root folder

Steps to run locally:
Create a .env file in backend/ and paste the variables

Open a new terminal
Enter backend directory - cd backend
 - npm install
 - npm run dev

Open a new terminal
Enter frontend directory - cd frontend
 - npm install
 - npm run dev
